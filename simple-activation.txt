SIMPLE PLATFORM ACTIVATION
==========================

Go to: https://tradingpost.appwrite.network
Press F12 to open console
Copy and paste this ONE LINE at a time:

1. Load Appwrite:
const script = document.createElement('script'); script.src = 'https://cdn.jsdelivr.net/npm/appwrite@15.0.0'; document.head.appendChild(script);

2. Wait 3 seconds, then run this:
const client = new Appwrite.Client(); client.setEndpoint('https://nyc.cloud.appwrite.io/v1').setProject('689bdee000098bd9d55c'); const account = new Appwrite.Account(client); account.getSession('current').catch(e => console.log('✅ Platform activated!'));

===========================

Just try to login with OAuth on your site:
